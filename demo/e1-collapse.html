e1-collapse

This element will collapse it's content when it's width is less than the supplied collapse width. A toggle element will be automatically inserted if not provided. If it is provided, be sure to also wrap your content in an element with a "e1-collapse-content" attribute. The toggle also requires a "e1-collapse-toggle" attribute.
