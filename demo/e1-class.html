e1-class

Evaluates the supplied value and adds/removes classes to the element. There are 2 ways to go about it:

  • You can supply a single or && separated string of bindings that will add their values as classes.
  • You can supply a single or && separated string of bindings, each with a conditional binding and a value binding separated by a ?. If the conditional binding is truthy, the value binding will be added as a class.
